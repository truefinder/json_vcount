# json_vcount

Total count of json key with value 


# Usage 
```
Usage: json_vcount.py [options] args

Options:
  -h, --help            show this help message and exit
  -f FILE, --file=FILE  ex) -f config.json
  -d DECODE, --decode=DECODE
                        none|urldecode|base64

```
